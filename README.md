📌 Project Overview

The Todo List Web App is a simple task management application that allows users to add, track, and delete tasks easily.

This project was built using Vanilla JavaScript to practice:

DOM manipulation
Event handling
Dynamic element creation
UI styling with CSS

This is my 1st Minor Project created while learning Web Development.

✨ Features

✔ Add new tasks dynamically
✔ Delete tasks instantly
✔ Mark tasks as completed using checkbox
✔ Input validation (prevents empty tasks)
✔ Smooth UI animations
✔ Neon-themed modern interface
